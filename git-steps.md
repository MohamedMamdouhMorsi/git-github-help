>   start new local repository 

```

```
> open your project 

```

```
### step 4 >  repository mange

> status

```
git status
```

> new file 

```
touch index.html
```
> add file to stage

```
git add index.html
```
> delete file from stage

```
git rm --cached index.html
```
### step 4 > commit

```

```

> chick your change on the log
